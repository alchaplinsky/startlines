Github
01 October, 2007
Development of the GitHub platform began. And in a year there were four full-time employees: Tom Preston-Werner, Chris Wanstrath, PJ Hyett, and Scott Chacon. And they topped 20,000 public repositories. GitHub, Inc. was originally known as Logical Awesome.
08 February, 2008
GitHub, a San Francisco company was founded as a way for open-source software writers in disparate locations to rapidly create new and better versions of their work. Work is stored, shared and discussed, based on the idea of a “pull request,” which is a suggestion to the group for some accretive element, like several lines of code, to be “pulled,” or added, to a project.
10 April, 2008
GitHub is a Web-based Git repository hosting service, which offers all of the distributed revision control and source code management (SCM) functionality of Git as well as adding its own features was launched. GitHub provides a Web-based graphical interface and desktop as well as mobile integration. It also provides access control and several collaboration features such as bug tracking, feature requests, task management, and wikis for every project. The site was launched by Tom Preston-Werner, Chris Wanstrath, and PJ Hyett after it had been made available for a few months prior as a beta period.
24 February, 2009
GitHub team members announced, in a talk at Yahoo! headquarters, that within the first year of being online, GitHub had accumulated over 46,000 public repositories, 17,000 of which were formed in the previous month alone. At that time, about 6,200 repositories had been forked at least once and 4,600 had been merged.
05 July, 2009
GitHub reached 100,000 users.
27 July, 2009
Tom Preston-Werner announced that GitHub had grown to host 90,000 unique public repositories, 12,000 having been forked at least once, for a total of 135,000 repositories.
01 July, 2010
Ruby and JavaScript are the most popular languages on GitHub, with 19 and 17 percent of the hosted code respectively, but there are currently projects in over 50 languages on the service: everything from Visual Basic to Go.
25 July, 2010
GitHub announced that it hosts 1 million repositories. It was confirmed by Scott Chacon, VP of Research and Development at GitHub. Approximately 60 percent of these projects are full repositories – that is, shared folders with code spread across multiple files – while the remaining 40 percent are “gists”, or short code snippets contained in a single file.
09 September, 2010
97 Sunlight software projects hosted on GitHub. GitHub gets all the credit. Much of the service’s impressiveness comes from its underlying revision control system, Git.
20 April, 2011
There were 2 million repositories in GitHub.
02 June, 2011
Github became the most popular open source forge, having surpassed Sourceforge, Google Code and Microsoft's CodePlex in total number of commits.
01 July, 2012
GitHub received $100 million from the venture capital firm Andreessen Horowitz.
09 July, 2012
Peter Levine, general partner at GitHub's investor Andreessen Horowitz, stated that GitHub had been growing revenue at 300% annually since 2008 "profitably nearly the entire way".
01 August, 2012
The code for "We the People" and the mobile apps, White House for iOS and White House for Android were released.
28 December, 2012
The number of users jumped to 2.8 million from 1.2 million. Companies pay to use GitHub, and it has become an exceptionally popular way for people to do all kinds of software work. The number of “repositories” — containing code, its documentation, images associated with a project or other work — increased to 4.6 million from 1.7 million last year.
16 January, 2013
GitHub reached 3 million users mark and was then hosting more than 5 million repositories.
09 May, 2013
The White House drafted and released the official Open Data Policy of the United States on the service. Project Open Data is a policy that instructs government agencies to “manage their data, and information more generally, as an asset from the start.” And not just to manage it for monetary or political gain but to manage it for the public good — the document says that government data should, wherever possible, be released “to the public in a way that makes it open, discoverable, and usable.”
23 May, 2013
GitHub had 3.5 million users hosting some 6 million "repositories" of stored software projects. Some of those users are big players like Lockheed Martin, Twitter, Facebook and The White House; enterprise users include Fortune 500 companies, large government agencies and universities. Revenue has increased 300 percent annually since GitHub's founding in 2008, and the staff now numbers 158.
19 July, 2013
The newest feature of its business model is to rent out a version to companies they can use internally. GitHub was listed as one of the ways to fix company’s stifling bureaucracy.
07 August, 2013
GitHub users work on seven million projects. You can find projects written in almost any programming language in existence, from common Web development languages like P.H.P. and JavaScript to the ancient low-level language Fortran, and even LOLCODE, an absurdist programming syntax jokingly assembled from LOLcat captions.
23 December, 2013
There were 10 million repositoriesin GitHub.
17 March, 2014
GitHub programmer Julie Ann Horvath alleged that Preston-Werner and his wife Theresa engaged in a pattern of harassment against her that led to her leaving the company. After an internal investigation, GitHub CEO and cofounder Chris Wanstrath issued a statement on the company blog personally apologizing to Horvath.
16 May, 2014
The second annual Crunchies announced that GitHub is a winner in Best Bootstrapped Startup
17 July, 2014
GitHub, Inc. introduced a layer of middle management. Before this Github didn't have middle managers, there was self-management.
03 December, 2014
Russia’s regulator RosComNadzor has blocked GitHub after the popular software and coding collaboration platform was found to be hosting content related to suicide. The “block” effectively amounts to an order to ISPs to restrict access to the site. And because GitHub works on HTTPS, providers can only comply by restricting access to the entire site, rather than individual pages.
31 December, 2014
GitHub was blocked in India (along with 31 other Websites) over pro-ISIS content posted by users. On 10 January 2015, GitHub was unblocked.
26 March, 2015
The largest DDoS (distributed denial of service) attack in github.com's history. The attack began around 2AM UTC on Thursday, March 26, and involved a wide combination of attack vectors. These included every vector in previous attacks as well as some sophisticated new techniques that used the web browsers of unsuspecting, uninvolved people to flood github.com with high levels of traffic.
30 March, 2015
GitHub reports having over 9 million users and over 21.1 million repositories, making it the largest host of source code in the world.
01 June, 2015
GitHub opened an office in Japan. That is its first office outside of the U.S.
25 July, 2015
GitHub raised $250 million in funding in a round led by Sequoia Capital. The round valued the company at approximately $2 billion.
01 September, 2015
GitHub has 324 employees and 10 million users. Architects, musicians, city governments, builders and others are currently using GitHub to share and collaborate on projects beyond software code.
01 September, 2015
GitHub is adding users at the rate of 10,000 per weekday.
02 October, 2015
After months of letting early adopters kick the tires, Git Large File Storage (Git LFS) has reached a 1.0 milestone and from now on is available to all repositories on GitHub.com.
